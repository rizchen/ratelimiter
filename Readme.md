### Step 1: Download file
```
$ git clone https://github.com/rizchen/ratelimiter.git
```

### Step 2: to the folder 
```
$ cd /ratelimiter
```

### Step 3: Build Docker image
```
$ docker-compose build
```


### Step 4: Run service
```
$ docker-compose up -d
```

### Step5: Stop service
```
$ docker-compose down
```
